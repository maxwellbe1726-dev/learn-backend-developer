Plan de 8 meses

Ruta tipo carretera: aprende construyendo JanSahayak

Un proyecto backend incremental: de consola Java a API segura con PostgreSQL, Docker, CI/CD e integración IA.

¿Qué es JanSahayak?

JanSahayak (del hindi "asistente del pueblo") es el proyecto integrador que construyes durante toda la ruta: un sistema backend de gestión y consulta de trámites ciudadanos. Empieza como una CLI en Java y crece mes a mes hasta convertirse en una API REST segura con PostgreSQL, Docker, CI/CD e integración de IA. Cada módulo añade una capa real al mismo proyecto, así que al final tendrás una sola aplicación completa para tu portafolio.

Consideraciones profesionales

Criterio arquitectónico

El valor de un ingeniero backend no radica en la memorización de código boilerplate, sino en su criterio arquitectónico para estructurar modularmente sistemas escalables y supervisar de forma experta la correctitud y seguridad de soluciones creadas con asistentes de IA.

Portafolio real

Documenta decisiones, diagramas, endpoints, pruebas, Docker Compose y análisis de complejidad. Tu GitHub debe contar una historia técnica.

Inglés técnico

La comunicación bilingüe multiplica oportunidades remotas y nearshore. Practica explicar arquitectura, trade-offs y debugging en inglés.

Producción sobre teoría aislada

Aprende cada tema con entregables. La base algorítmica debe verse aplicada a búsquedas, colas, cachés, índices y optimización real.
